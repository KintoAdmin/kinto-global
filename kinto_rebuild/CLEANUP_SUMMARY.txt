Kinto build_v70 cleaned package

Purpose:
This cleaned package removes non-essential local and generated content from the uploaded build package so the baseline is easier to store, share, and restore.

Removed categories:
- .env.local (local secrets / machine-specific config)
- .next (generated Next.js build cache/output)
- node_modules (reinstallable dependencies)
- qa-artifacts (test outputs)
- runtime_artifacts (generated reports and runtime files)
- legacy QA/E2E packs and duplicate QA docs
- old build notes / patch notes / release-note debris
- tsconfig.tsbuildinfo and stray testfile

Kept:
- source code
- python engine
- scripts
- data libraries
- supabase migrations
- package manifests
- README.md
- docs/SETUP.md
- .env.example

Expected setup after unzip:
1. Copy or create a fresh .env.local from .env.example / your real env values
2. Run npm install
3. Run npm run dev and npm run py:dev
