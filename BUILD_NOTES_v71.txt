INSTALL STEPS (same as v70)
---------------------------
1. Extract zip to your project folder
2. Copy your .env.local into the root
3. Confirm migrations 001-004 are applied in Supabase
4. npm install
5. pip install -r python_engine/requirements.txt
6. npm run seed:reference (if not already done)
7. npm run dev:full
8. Open http://localhost:3000
